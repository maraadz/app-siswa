## ğŸ› ï¸ Tech Stack & Framework
Aplikasi ini menggunakan teknologi terbaru untuk performa maksimal:

* **Core:** [React.js](https://react.dev/) + [Vite](https://vitejs.dev/) (Build tool super cepat).
* **Language:** [TypeScript](https://www.typescriptlang.org/) (Menjamin kode lebih stabil dan minim error).
* **Styling:** [Tailwind CSS](https://tailwindcss.com/) (Custom UI dengan sistem utility-first).
* **Icons:** [Lucide React](https://lucide.dev/) & [Ionicons](https://ionic.io/).
* **State:** [Zustand](https://github.com/pmndrs/zustand) (Manajemen data user dan filter yang ringan).
* **API:** [Axios](https://axios-http.com/) dengan interceptor untuk keamanan token.
